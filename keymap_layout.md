## キー略語説明

| 略語 | 意味 |
|------|------|
| MO1-3 | モーメンタリーレイヤー1-3 (押している間だけ有効) |
| LClk | 左クリック |
| RClk | 右クリック |
| MClk | 中クリック |
| MB 1-3 | マウスボタン1-3 |
| BSpc | バックスペース |
| DEL | デリート |
| BT | Bluetooth |
| CLR | クリア |
| NXT | 次 |
| PRV | 前 |
| DISC | 切断 |
| 半/全 | 半角/全角切替 |

## レイヤー構造

```
Layer 0 (デフォルト)
  │
  ├── MO1 → Layer 1 (シンボル/編集)
  ├── MO2 → Layer 2 (数字/カーソル)
  ├── MO3 → Layer 3 (ファンクション/BT)
  │
  └── コンボ
      ├── MO2+MO3 → Layer 4 (マウス)
      └── MO3+Space → Layer 5 (スクロール)

Layer 6 (システム) - 直接アクセス方法は定義されていません
```
